# 📊 机器学习与 SHAP 可解释性分析工具 —— 让 AI 决策「看得见」

> 一款基于 Streamlit 的零代码机器学习分析平台，支持分类/回归建模，内置 SHAP 可解释性分析，助你快速理解模型「黑箱」背后的逻辑。

---

## 一、为什么需要它？

在机器学习项目中，我们常常面临这样的困境：

- **模型效果不错，但说不清为什么** —— 业务方追问「哪些因素最重要？」时，只能含糊其辞
- **想尝试 SHAP 分析，却要写大量代码** —— 从数据预处理到模型训练，再到 SHAP 可视化，每一步都要自己实现
- **不同模型、不同任务要反复切换脚本** —— 分类、回归、多种算法，维护成本高

**本工具** 将数据探索、模型训练、评估与 SHAP 可解释性分析整合到一个 Web 界面中，**无需编写代码**，上传数据即可完成全流程分析，并输出可直接用于论文、报告的矢量图。

---

## 二、核心功能一览

### 1. 数据接入与探索

| 功能 | 说明 |
|------|------|
| **多格式支持** | 支持 CSV、Excel（.xlsx / .xls）文件上传 |
| **示例数据** | 内置分类、回归示例数据，可快速体验 |
| **数据概览** | 基本统计、分布直方图、特征相关性热力图 |
| **缺失值处理** | 均值 / 中位数 / 众数填充，或删除含缺失行 |
| **特征选择** | 方差阈值、相关性过滤、基于模型 Top-K 筛选 |

从原始数据到建模特征，在界面中即可完成清洗与筛选，无需写 `pandas` 代码。

---

### 2. 任务类型与模型

**分类任务**（二分类 / 多分类）支持：

- 随机森林、逻辑回归、梯度提升、SVM、KNN  
- XGBoost、LightGBM（需安装对应库）

**回归任务** 支持：

- 随机森林、岭回归、梯度提升、SVR、KNN  
- XGBoost、LightGBM（需安装对应库）

**高级选项**：

- **模型对比**：一次训练多个模型，自动选出最佳
- **超参数调优**：网格搜索 / 随机搜索，自动寻找最优参数
- **K 折交叉验证**：评估模型稳定性
- **特征标准化**：适合逻辑回归、岭回归等线性模型

---

### 3. 模型评估

**分类任务**：

- 准确率、F1 分数、分类报告
- 混淆矩阵热力图
- ROC 曲线、PR 曲线（二分类 / 多分类）

**回归任务**：

- R²、RMSE、MAE
- 残差图、预测 vs 真实散点图

所有指标与图表均在界面中展示，便于快速判断模型表现。

---

### 4. SHAP 可解释性分析（核心亮点）

SHAP（SHapley Additive exPlanations）基于博弈论中的 Shapley 值，为每个特征分配对预测的贡献。本工具提供 **10 类 SHAP 可视化**：

| 图表类型 | 用途 |
|----------|------|
| **SHAP 摘要图** | 条形图 / 蜂群图，展示各特征的整体重要性及正负贡献 |
| **特征依赖图** | 某特征取值与 SHAP 值的关系，发现非线性效应 |
| **交互图** | 主特征与另一特征的交互效应，颜色表示交互特征取值 |
| **SHAP 交互值** | 树模型专用，展示特征间交互贡献矩阵 |
| **热力图** | 按样本展示各特征 SHAP 值，样本按聚类排序 |
| **决策图** | 各样本的累积 SHAP 值变化，每条线代表一个样本 |
| **小提琴图** | 各特征 SHAP 值的分布（violin / layered_violin） |
| **瀑布图** | 单样本各特征对预测的贡献分解 |
| **力图** | 单样本各特征对预测的推动 / 拉低作用 |
| **特征重要性对比** | SHAP 与模型自带 `feature_importances_` 的对比 |

**多分类** 时，可切换解释目标类别；**树模型** 自动使用 TreeExplainer，**线性模型** 使用 LinearExplainer，无需手动配置。

---

### 5. 图形导出与定制

- **格式**：PNG、PDF、SVG、EPS  
- **矢量格式**（PDF / SVG / EPS）可在 Illustrator、Inkscape 中编辑，适合论文、报告
- **批量导出**：一键打包所有 SHAP 图为 ZIP 下载
- **自定义**：可调 DPI、字体大小，适配不同展示需求

---

### 6. 界面与体验

- **暗色 / 亮色主题**：侧边栏一键切换
- **数据缓存**：训练与 SHAP 结果缓存，切换参数时加快响应
- **响应式布局**：宽屏展示，图表与表格自适应

## 四、快速开始

**环境要求**：Python 3.9+

```bash
cd streamlit_ML_SHAP
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
streamlit run app.py
```

浏览器访问 `http://localhost:8501` 即可使用。

**推荐流程**：

1. 侧边栏选择数据来源（上传文件或示例数据）、任务类型、模型
2. 主界面选择目标列，完成数据探索与预处理
3. 查看模型评估指标与图表
4. 在 SHAP 分析中切换不同图表类型，选择格式后下载
