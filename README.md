# 机器学习与 SHAP 可解释性分析（Streamlit）

基于 Streamlit 的机器学习建模与 SHAP 可解释性分析应用，支持上传自有数据或使用内置示例数据，完成分类/回归建模并可视化特征重要性及单样本解释。

## 功能概览

- **数据**：上传 CSV/Excel，或使用内置分类/回归示例数据
- **数据探索与预处理**：数据概览（统计、分布直方图、相关性热力图）、缺失值处理（均值/中位数/众数填充或删除）、特征选择（方差阈值、相关性过滤、基于模型 Top-K）
- **任务类型**：二分类/多分类、回归
- **模型**：随机森林、逻辑回归（分类）；随机森林、岭回归（回归）
- **评估**：准确率、F1、分类报告、混淆矩阵（分类）；R²、RMSE、MAE（回归）
- **SHAP 分析**：
  - 摘要图（条形图 / 蜂群图）
  - 特征依赖图
  - 热力图（样本聚类排序）
  - 决策图（累积 SHAP 变化）
  - 小提琴图（SHAP 值分布）
  - 瀑布图（单样本贡献分解）
  - 力图（单样本推动/拉低作用）

## 环境与安装

建议使用 Python 3.9+ 与虚拟环境：

```bash
cd streamlit_ML_SHAP
python -m venv .venv
.venv\Scripts\activate   # Windows
# source .venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
```

## 运行方式

```bash
streamlit run app.py
```

浏览器会打开 `http://localhost:8501`。若需指定端口：

```bash
streamlit run app.py --server.port 8502
```

## 使用步骤

1. **侧边栏**：选择数据来源（上传文件 / 示例数据）、任务类型（分类/回归）、模型、测试集比例、是否标准化。
2. **主界面**：若为上传文件，选择目标列；查看数据预览与特征列。
3. 程序自动划分训练/测试集、训练模型并展示评估指标。
4. 在 **SHAP 可解释性分析** 中切换标签页查看：
   - **SHAP 摘要图**：各特征对预测的整体贡献（条形图或蜂群图）。
   - **特征依赖图**：选择某一特征，查看其取值与 SHAP 值的关系。
   - **交互图**：展示特征间的交互效应，可选择主特征与着色特征（自动/指定/无）。
   - **热力图**：按样本展示各特征 SHAP 值，样本按聚类排序。
   - **决策图**：展示各样本的累积 SHAP 值变化，可调节样本数。
   - **小提琴图**：各特征 SHAP 值的分布（violin / layered_violin）。
   - **瀑布图**：选择测试集样本索引，查看该样本各特征对预测的贡献分解。
   - **力图**：选择测试集样本索引，查看该样本各特征对预测的推动/拉低作用。
5. **图形保存**：侧边栏选择保存格式（PNG / PDF / SVG / EPS），每个 SHAP 图下方提供「下载图形」按钮。PDF、SVG、EPS 为矢量格式，可在 Illustrator、Inkscape 等软件中编辑。

## 依赖说明

- `streamlit`：Web 界面
- `pandas`、`numpy`：数据处理
- `scikit-learn`：模型与评估
- `shap`：SHAP 值计算与图表
- `matplotlib`：绘图后端
- `openpyxl`：Excel 读取

## 注意事项

- 仅使用数值型列作为特征；非数值目标在分类任务中会自动标签编码。
- 逻辑回归、岭回归建议勾选“对特征做标准化”。
- SHAP 对线性模型使用 `LinearExplainer`，对树模型使用 `TreeExplainer`；若某模型解释失败，界面会提示并跳过 SHAP 部分。
